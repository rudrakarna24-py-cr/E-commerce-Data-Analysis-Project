# E-commerce Data Analysis Project

## Overview
Performed Exploratory Data Analysis (EDA) on a diversified e-commerce dataset using Python.

## Tools Used
- Python (Pandas, NumPy, Matplotlib)
- Google Colab

## Key Work
- Data Cleaning (missing values, duplicates)
- Feature Engineering (final price after discount)
- Data Analysis using groupby, ranking (SQL-like operations)
- Data Visualization (dashboard, charts)

## Insights
- Identified high-performing product categories
- Analyzed return rates and pricing trends
- Studied customer behavior patterns

## Output
- EDA Dashboard
- Business insights from data
